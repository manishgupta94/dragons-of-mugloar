# Dragons of Mugloar #

[Dragons of Mugloar](http://www.dragonsofmugloar.com)

Solution implemented with Java.

#### Dependencies ####
   * Java 8
   * Maven
   * GSON
   * JAXB2
   * Apache Commons
   
## Installation ##
   * Clone or fork the repo
   * `mvn install` for install or `mvn clean package` for build
